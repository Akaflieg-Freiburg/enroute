\doxysubsection{Detailed Description}
\doxylink{classFileFormats_1_1TIFF}{TIFF} support. 

This class reads \doxylink{classFileFormats_1_1GeoTIFF}{Geo\+TIFF} files. It extracts image dimension as well as the \doxylink{classFileFormats_1_1TIFF}{TIFF} fields. It does not read the raster data.
