\doxysubsection{Detailed Description}
METAR/\+TAF decoder. 

This class takes \doxylink{classWeather_1_1METAR}{METAR} or \doxylink{classWeather_1_1TAF}{TAF} messages in raw form and converts them to human-\/readable, translated text. This class is not meant to be used directly. Instead, use the classes \doxylink{classWeather_1_1METAR}{Weather\+::\+METAR} or \doxylink{classWeather_1_1TAF}{Weather\+::\+TAF}.
