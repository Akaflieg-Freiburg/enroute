Detailed Description

Traffic opponents.

Objects of this class represent traffic opponents, as detected by FLARM and similar devices. Instances of this class will be generated by the Navigation::FLARMAdaptor class. Consumers of this class will never have to set or construct instances of the class themselves

◆ alarmLevel

int Traffic::FLARMWarning::alarmLevel
read

Alarm Level, as reported by FLARM.

This is the alarm level associated with the traffic object. Alarm levels are not computed by this class, but by the FLARM device that reports the traffic. This is an integer in the range 0, …, 3 with the following meaning.

  • 0 = no alarm (also used for no-alarm traffic information)
  • 1 = alarm, 13-18 seconds to impact
  • 2 = alarm, 9-12 seconds to impact
  • 3 = alarm, 0-8 seconds to impact
