\doxysubsection{Detailed Description}
Geographic position. 

This class is a thin wrapper around QGeo\+Position\+Info. It exports the data from QGeo\+Position\+Info in a way that can be read from QML. In addition to QGeo\+Position\+Info, the class also contains information about the pressure altitude.
