\doxysubsection{Detailed Description}
Implementation of QHttp\+Engine\+::\+Handler that serves mbtile files. 

This is a helper clas for \doxylink{classGeoMaps_1_1TileServer}{Tile\+Server}. It gathers a set of MBTiles files. The method \doxylink{classGeoMaps_1_1TileHandler_a58ebe88bd1a4b517337ae5d71b211975}{process()} takes the path of an incoming HTTP request and uses a QHttp\+Server\+Responder to reply with appropriate tile data, and with Tile\+JSON (following the Tile\+JSON Specification 2.\+2.\+0 found in \href{https://github.com/mapbox/tilejson-spec/tree/master/2.2.0}{\texttt{ https\+://github.\+com/mapbox/tilejson-\/spec/tree/master/2.\+2.\+0}}).
