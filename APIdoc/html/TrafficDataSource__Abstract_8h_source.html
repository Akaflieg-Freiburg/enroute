520 /* This method expects exactly one line containing a valid FLARM/NMEA
521 * sentence. This is a string typically looks like
522 * "$PFLAA,0,1587,1588,40,1,AA1237,225,,37,-1.6,1*7F". The method
523 * interprets the string and updates the properties and emits signals as
524 * appropriate. Invalid strings are silently ignored.
525 */
