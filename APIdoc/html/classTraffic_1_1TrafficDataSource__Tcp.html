Detailed Description

Traffic receiver: TCP connection to FLARM/NMEA source.

This class connects to a traffic receiver via a TCP connection. It expects to find a receiver at the specifed IP-Address and port that emits FLARM/NMEA sentences.

In most use cases, the connection will be established via the device's WiFi interface. The class will therefore try to lock the WiFi once a heartbeat has been detected, and release the WiFi at the appropriate time.
