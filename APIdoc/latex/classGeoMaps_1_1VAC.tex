\doxysubsection{Detailed Description}
Visual approach chart. 

This class represents visual approach charts. It stores the following data items.


\begin{DoxyItemize}
\item The name of the visual approach chart.
\item The name of a raster image file.
\item Geographic coordinates for the four corners of the raster image 
\end{DoxyItemize}

Geo\+Maps\+::\+VAC\+::\+VAC (\begin{DoxyParamCaption}\item[{const QString \&}]{f\+Name}{}\end{DoxyParamCaption})}



Constructor. 

This class reads a georeferenced image file, where geographic data is encoded in one of the following two forms.


\begin{DoxyItemize}
\item The image file is a Geo\+TIFF file with embedded georeferencing information.
\item The file name is of the form "{}\+EDTF-\/geo\+\_\+7.\+739665\+\_\+48.\+076416\+\_\+7.\+9063883\+\_\+47.\+96452.\+jpg"{}
\end{DoxyItemize}

It attempt to extract the map name from the image file (if the image file is a Geo\+TIFF), or else from the file name. The raster data is not read, so that this constructor is rather lightweight.
