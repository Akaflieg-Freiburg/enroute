149 /* \brief Verbose description of waypoint properties
150 *
151 * This property holds a list of strings in meaningful order that describe
152 * the waypoint properties. This includes airport frequency, runway
153 * information, etc. The data is returned as a list of strings where the
154 * first four letters of each string indicate the type of data with an
