\doxysubsection{Detailed Description}
Intended flight route. 

This class represents an intended flight route. In essence, this class is little more than a list of waypoint and a number of methods that do the following.


\begin{DoxyItemize}
\item Expose the list of waypoints and legs to QML and allow some manipulation from there, such as adding or re-\/arranging waypoints.
\item Compute length and true course for the legs in the flight path, as well as a total length and expose this data to QML. 
\end{DoxyItemize}
