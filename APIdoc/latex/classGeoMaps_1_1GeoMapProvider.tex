\doxysubsection{Detailed Description}
Provides geographic information. 

This class works closely with data\+Management/\+Data\+Manager. It takes the data provided by the Data\+Manager, and serves it for use in Map\+Box\+GL powered maps. Additional data is served via the API.


\begin{DoxyItemize}
\item The class ensures that the currently available base maps are served via the embedded \doxylink{classGeoMaps_1_1TileServer}{Tile\+Server}.
\item The class generates a mapbox style file whose source element points to the URL of the embedded \doxylink{classGeoMaps_1_1TileServer}{Tile\+Server}. The style file automatically adjusts when raster maps or vector maps are installed.
\item All available aviation data is provided in \doxylink{classGeoMaps_1_1GeoJSON}{Geo\+JSON}.
\item Waypoints and airspaces are accessible via the API. 
\end{DoxyItemize}
