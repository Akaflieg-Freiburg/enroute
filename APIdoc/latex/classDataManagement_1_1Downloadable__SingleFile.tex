\doxysubsection{Detailed Description}
Downloadable object. 

This class represents a downloadable item, such as an aviation map file. The class is able to do the following.


\begin{DoxyItemize}
\item Download the file asynchronously from the server.
\item Check if a newer version of the file is available at the URL and update the file if desired.
\end{DoxyItemize}

The URL and the name of the local file are given in the constructor and cannot be changed. See the description of the method \doxylink{classDataManagement_1_1Downloadable__SingleFile_ac572ce2dd771329cfbf08c66cfd7335b}{start\+Download()} to see how downloads work.

Q\+\_\+\+INVOKABLE void Data\+Management\+::\+Downloadable\+\_\+\+Single\+File\+::start\+Download (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [override]}, {\ttfamily [virtual]}}



Initiate a download. 

Initiate an asynchronous download of the remote file. If a download is already in progress, nothing will happen. Otherwise, the following will take place.


\begin{DoxyEnumerate}
\item Data is retrieved from the remote server and stored in a temporary file. The signal \doxylink{classDataManagement_1_1Downloadable__SingleFile_ad3cab1f161975ff2d30f2950bc494e85}{download\+Progress()} will be emitted regularly.
\item In case of an error, the signal \doxylink{classDataManagement_1_1Downloadable__Abstract_a0d0cedd2c97765a34ead1b0553389499}{error()} is emitted and the download stops.
\item Optionally, the download can be stopped using the method stop\+File\+Download().
\end{DoxyEnumerate}

Once all data has been downloaded successfully to the temporary file, the process continues as follows.


\begin{DoxyEnumerate}
\item The signal about\+To\+Change\+Local\+File() is emitted. As the name suggests, this indicates that the local file is about to change and that it should not be used anymore.
\item A QLock\+File is created at \doxylink{classDataManagement_1_1Downloadable__SingleFile_a89315c6cc9e560d2836436e7e1b05872}{file\+Name()}+"{}.\+lock"{}
\item The local file is overwritten by the newly downloaded data.
\item The QLock\+File is removed
\item The signal file\+Changed() is emitted to indicate that the file is again ready to be used. 
\end{DoxyEnumerate}
