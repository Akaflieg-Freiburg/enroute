\doxysubsection{Detailed Description}
This class represents a weather station that issues \doxylink{classWeather_1_1METAR}{METAR} or \doxylink{classWeather_1_1TAF}{TAF} report. 

This is a very simple class that represents a weather station. Weather stations are uniquely identified by their ICAO code. Depending on available data, they hold pointers to the latest \doxylink{classWeather_1_1METAR}{METAR} and \doxylink{classWeather_1_1TAF}{TAF} reports.
