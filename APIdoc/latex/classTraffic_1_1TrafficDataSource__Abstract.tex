\doxysubsection{Detailed Description}
Base class for all traffic receiver data sources. 

This is an abstract base class for all classes that connect to a traffic receiver. In addition to the properties listed below, the class also emits imporant data via the signals barometric\+Altitude\+Updated, factor\+Without\+Position, factor\+With\+Position and warning. It contains methods to interpret FLARM and GDL90 data streams.
