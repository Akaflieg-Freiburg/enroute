Detailed Description

METAR/TAF decoder.

This class takes METAR or TAF messages in raw form and converts them to human- readable, translated text.

QString Meteorologist::Decoder::decodedText
read

Decoded text of the METAR/TAF message

This property holds the decoded text of the message, as a human-readable, rich text string. The text might change in responde to changes in user settings, and might also change by midnight (the text uses words such as 'tomorrow' whose meaning changes at the end of the day).
