\doxysubsection{Detailed Description}
Traffic receiver\+: UDP connection to GDL90 source. 

This class connects to a traffic receiver via a GDL90 connection. It expects to find a receiver at the specifed IP-\/\+Address and port that emits GDL90 messages.

In most use cases, the connection will be established via the device\textquotesingle{}s Wi\+Fi interface. The class will therefore try to lock the Wi\+Fi once a heartbeat has been detected, and release the Wi\+Fi at the appropriate time.
