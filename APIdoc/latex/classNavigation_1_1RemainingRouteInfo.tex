\doxysubsection{Detailed Description}
Info about remaining route. 

This class bundles information about the remainder of the present flight. This includes next waypoint, final waypoint, as well as distance, ETE and ETA.

Consumers of this class can expect the properties next\+WP\texorpdfstring{$\ast$}{*} to be valid if status equals On\+Route. The properties final\+WP\texorpdfstring{$\ast$}{*} need not be valid in this case. They are valid only if the next waypoint differs from the final waypoint of the route.
