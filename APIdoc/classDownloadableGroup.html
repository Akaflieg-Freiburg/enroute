Detailed Description

Manages a set of downloadable objects.

This convenience class collects signals and properties from a set of Downloadable objects, and forwards summarized information.

◆ addToGroup()

void DownloadableGroup::addToGroup ( Downloadable *  downloadable)

Adds a Downloadable to the group.

This method adds a Downloadable object to the group. The signals 'downloadingChanged' and 'updatableChanged' will be emitted if required.

The DownloadableGroup does not take ownership of the Downloadable, and it is safe to delete the Downloadable after it has been added, without removing it first. It is perfectly fine for Downloadable objects to be members of several groups.

Parameters
downloadablePointer to the Downloadable to be added.
