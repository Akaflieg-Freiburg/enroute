Detailed Description

Waypoint, such as an airfield, a navaid station or a reporting point.

This class represents a waypoint feature of a GeoJSON file, as described in here. In essence, it contains two bits of data:

  • The coordinate of the waypoint.
  • A QMap<QString, QVariant> that represents the waypoint properties found in the GeoJSON file.
