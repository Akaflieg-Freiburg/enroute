This class describes a connection to a traffic data receiver. It exposes connection properties in a format that is suitable for QML.

Equality of connection. 

This test for equality is not strict. It returns \textquotesingle{}true\textquotesingle{} if the two connection infos describe the same connection. Still, they can differ in aspects such as the connection name or description. This is possible, e.\+g., for Bluetooth devices whose name was not know when one of the Connection\+Infos was constructed.
