\doxysubsection{Detailed Description}
\doxylink{classWeather_1_1WeatherDataProvider}{Weather\+Data\+Provider}, weather service manager. 

This class retrieves METAR/\+TAF weather reports from the "{}\+Aviation Weather \+Center"{} at aviationweather.\+com, for all weather stations that are within 75nm from the last-\/known user position or current route. The reports can then be accessed via the property "{}weather\+Stations"{} and the method find\+Weather\+Station. The \doxylink{classWeather_1_1WeatherDataProvider}{Weather\+Data\+Provider} class honors Global\+Settings\+::accepted\+Weather\+Terms() and will initiate a download only if the user agreed to the privacy warning.

Once constructed, the \doxylink{classWeather_1_1WeatherDataProvider}{Weather\+Data\+Provider} class will regularly perform background updates to retrieve up-\/to-\/date information. It will update the list of known weather stations and also the METAR/\+TAF reports for the weather stations. The class checks regularly for outdated \doxylink{classWeather_1_1METAR}{METAR} and \doxylink{classWeather_1_1TAF}{TAF} reports and deletes them automatically, along with those Weather\+Stations that no longer contain any report.

In order to avoid loss of data when the app is accidently closed in-\/flight, the class stores all weather data at destruction and at regular intervals, and reads the data back in on construction.
