Detailed Description

Watches a group of Downloadable objects.

This convenience class collects signals and properties from a set of Downloadable objects, and forwards summarized information.

◆ localFileContentChanged_delayed

void DownloadableGroupWatcher::localFileContentChanged_delayed ( )
signal

Emitted some time after the content of one of the local files changes.

This signal is in principle identical to localFileContentChanged(), but is emitted with a delay. The DownloadableGroupWatch waits with the emission of this signal for two seconds. In addition it waits until there are no running download processes anymore.

◆ QPointer

QList DownloadableGroupWatcher::QPointer
read

List of Downloadables in this group.

List of Downloadable objects in this group that have local files.

This property holds the list of Downloadable objects in the group. The Downloadable objects are sorted alphabetically in ascending order, first by section() and then secondly by file name. The nullptr is never contained in the list.
