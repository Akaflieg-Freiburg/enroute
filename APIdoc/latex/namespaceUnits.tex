\doxysubsection{Detailed Description}
Conversion between units used in aviation. 

Wrapper around size\+\_\+t, to make it available to QML.

This class contains a few classes that help with the conversion of units, as used in aviation. It contains a few subclasses that computation with angles, speeds, distances, etc, without the need to worry about units.

This is a trivial C++ and QML value type, with a single member of type size\+\_\+t. Conversion methods guarantee that this class can be used in C++ as a drop-\/in replacement for size\+\_\+t. In QML, the class can be used to work to size\+\_\+t, which is otherwise not available.
