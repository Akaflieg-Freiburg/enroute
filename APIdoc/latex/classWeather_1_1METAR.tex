\doxysubsection{Detailed Description}
\doxylink{classWeather_1_1METAR}{METAR} report. 

This class contains the data of a \doxylink{classWeather_1_1METAR}{METAR} or SPECI report and provided a few methods to access the data. Instances of this class are provided by the \doxylink{classWeather_1_1WeatherDataProvider}{Weather\+Data\+Provider} class; there is no way to construct valid instances yourself.
