\doxysubsection{Detailed Description}
Abstract base class for all classes that provide geographic position information. 

This is the base class for all classes that provide geographic position information. The information is exposed via two properties, position\+Info and pressure\+Altitude.

The property status\+String gives more information about the status of the source
