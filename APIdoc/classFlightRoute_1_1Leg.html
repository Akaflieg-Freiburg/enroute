Constructs a flight route with given start and end point.

The Waypoint start and end are copied, so that this class does not make any assumption about the lifetome of *start and *end

Parameters
startPointer to the starting point
endPointer to the end point
aircraftPointer to aircraft info that is used in route computations. The aircraft object to supposed to exist throughout the liftime of this object.
windPointer to wind info that is used in route computations. The wind object to supposed to exist throughout the liftime of this object.
parentThe standard QObject parent pointer.
