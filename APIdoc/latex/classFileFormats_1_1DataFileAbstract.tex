\doxysubsection{Detailed Description}
Base class for file reading classes. 

This is an abstract base class for all classes that read data files, such as \doxylink{classFileFormats_1_1CUP}{CUP}, Geo\+JSON, \doxylink{classFileFormats_1_1GeoTIFF}{Geo\+TIFF}, â€¦. Implementations should adhere to the following conventions\+:


\begin{DoxyItemize}
\item The constructor takes a file name as an argument and reads the file, which might be quite expensive.
\item There is a static method \textquotesingle{}mime\+Types\textquotesingle{} that describes the mime types of the files that can be opened with this class. 
\end{DoxyItemize}
