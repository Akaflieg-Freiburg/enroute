\doxysubsection{Detailed Description}
\doxylink{classFileFormats_1_1GeoTIFF}{Geo\+TIFF} support. 

This class reads \doxylink{classFileFormats_1_1GeoTIFF}{Geo\+TIFF} files, as specified here\+: \href{https://gis-lab.info/docs/geotiff-1.8.2.pdf}{\texttt{ https\+://gis-\/lab.\+info/docs/geotiff-\/1.\+8.\+2.\+pdf}}

It extracts bounding box coordinates, as well as the name of the file. This class does not read the raster data. \doxylink{classFileFormats_1_1GeoTIFF}{Geo\+TIFF} is a huge and complex standard, and this class is definitively not able to read all possible valid \doxylink{classFileFormats_1_1GeoTIFF}{Geo\+TIFF} files. We restrict ourselves to files that appear in real-\/world aviation.
