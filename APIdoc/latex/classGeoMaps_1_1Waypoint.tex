\doxysubsection{Detailed Description}
\doxylink{classGeoMaps_1_1Waypoint}{Waypoint}, such as an airfield, a navaid station or a reporting point. 

This class represents a waypoint. The properties stored in this class correspond to the feature of the \doxylink{classGeoMaps_1_1GeoJSON}{Geo\+JSON} files that are used in Enroute, as described \href{https://github.com/Akaflieg-Freiburg/enrouteServer/wiki/GeoJSON-files-used-in-enroute-flight-navigation}{\texttt{ here}}.
