\doxysubsection{Detailed Description}
Utility class for databases in \doxylink{classFileFormats_1_1MBTILES}{MBTILES} format. 

\doxylink{classFileFormats_1_1MBTILES}{MBTILES} contain tiled map data. Internally, \doxylink{classFileFormats_1_1MBTILES}{MBTILES} are SQLite databases whose schema is specified here\+: \href{https://github.com/mapbox/mbtiles-spec}{\texttt{ https\+://github.\+com/mapbox/mbtiles-\/spec}} This class handles \doxylink{classFileFormats_1_1MBTILES}{MBTILES} and allows easy access to the data.
