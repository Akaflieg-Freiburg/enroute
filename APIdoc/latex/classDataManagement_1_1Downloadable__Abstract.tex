\doxysubsection{Detailed Description}
Abstract base class for \doxylink{classDataManagement_1_1Downloadable__SingleFile}{Downloadable\+\_\+\+Single\+File} and \doxylink{classDataManagement_1_1Downloadable__MultiFile}{Downloadable\+\_\+\+Multi\+File}.

\begin{DoxyNote}{Note}
Subclasses need to set the protected member m\+\_\+content\+Type in their constructors. 
\end{DoxyNote}
