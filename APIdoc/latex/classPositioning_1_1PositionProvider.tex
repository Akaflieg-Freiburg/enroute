\doxysubsection{Detailed Description}
Central Position Provider. 

This class collects position data from the various sources (satellite, network, traffic receiver, …) chooses the best available source and exposes the data to QML and other parts of the program.

There exists one static instance of this class, which can be accessed via the method global\+Instance(). No other instance of this class should be used.

Data from the standard operating system data source (typically\+: satnav or wifi) is only provided after \doxylink{classPositioning_1_1PositionProvider_ad3edcc87d04847b13ee547fd0fb16db4}{start\+Updates()} has been called.

The methods in this class are reentrant, but not thread safe.
