\doxysubsection{Detailed Description}
Interface to platform-\/specific file exchange functionality. 

This pure virtual class is an interface to file exchange functionality that need platform-\/specific code to operate. The files File\+Exchange\+\_\+\+XXX.(h\texorpdfstring{$\vert$}{|}cpp) implement a child class \doxylink{classPlatform_1_1FileExchange}{File\+Exchange} that contains the actual implementation.

If supported by the platform, child classes need to react to requests by the platform to open a file (e.\+g. a Geo\+JSON file containing a flight route). Once a request is received, the method process\+File\+Request() should be called.
