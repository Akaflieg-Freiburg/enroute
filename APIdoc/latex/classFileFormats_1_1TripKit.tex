\doxysubsection{Detailed Description}
Trip Kit. 

This class implements a Trip Kit reader. Trip kits are ZIP files that contain georeferenced VACs and/or other files that describe the georeferencing. At present, this class supports trip kits in two different formats\+:


\begin{DoxyItemize}
\item Trip\+Kits as produced by the "{}\+AIP Browser DE"{}, as specified here\+: \href{https://mpmediasoft.de/products/AIPBrowserDE/help/AIPBrowserDE\%20-\%20Trip-Kit-Spezifikation.html}{\texttt{ https\+://mpmediasoft.\+de/products/\+AIPBrowser\+DE/help/\+AIPBrowser\+DE\%20-\/\%20\+Trip-\/\+Kit-\/\+Spezifikation.\+html}}
\item ZIP file that contain image files of the form "{}base\+Name-\/geo\+\_\+7.\+739665\+\_\+48.\+076416\+\_\+7.\+9063883\+\_\+47.\+96452.\+xxx"{} or "{}base\+Name\+\_\+7.\+739665\+\_\+48.\+076416\+\_\+7.\+9063883\+\_\+47.\+96452.\+xxx"{}, where xxx is an ending such as jpg, png, webp. 
\end{DoxyItemize}
